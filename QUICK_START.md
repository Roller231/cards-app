# Быстрый старт - Новая система оплаты

## 1. Запуск приложения

```powershell
cd d:\GitPR\cards-app
docker compose -f docker-compose.dev.yml up --build
```

Подождите, пока все контейнеры запустятся (обычно 1-2 минуты).

## 2. Вход как dev-пользователь

1. Откройте браузер: `http://localhost:8080`
2. Нажмите **Ctrl+Shift+R** (жёсткий перезапуск)
3. Нажмите кнопку **"Dev auth"**
4. Вы вошли как `dev_user` с балансом **$1000**

## 3. Настройка цены в админке

1. Откройте: `http://localhost:8080/admin`
2. Логин: `admin` / Пароль: `admin123`
3. Перейдите в **Settings**
4. Найдите и измените:
   - **CARD_ISSUANCE_PRICE_USD** = `10.0` (цена выпуска карты)
5. Нажмите **Save**

## 4. Тестирование выпуска карты

1. Вернитесь на главную: `http://localhost:8080`
2. Нажмите **"Выпустить карту"**
3. Выберите тип карты (любой)
4. Вы увидите:
   - **Стоимость выпуска**: $10.00
   - **Начальный баланс карты**: $0.00
5. Выберите **СБП** как способ оплаты
6. Нажмите **"Оформить карту"**
7. Подтвердите выпуск

## Что происходит внутри

1. С баланса dev_user списывается **$10** (CARD_ISSUANCE_PRICE_USD)
2. Карта выпускается с **нулевым балансом**
3. Пользователь может пополнить карту после выпуска
4. Вся сумма $10 - ваша прибыль

## Проверка логов

```powershell
# Логи backend
docker compose -f docker-compose.dev.yml logs -f backend

# Логи всех сервисов
docker compose -f docker-compose.dev.yml logs -f
```

## Остановка

```powershell
docker compose -f docker-compose.dev.yml down
```
